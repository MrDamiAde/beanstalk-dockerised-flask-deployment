This project demosntrates my ability to deploy a flask appication on AWS Elastic Beanstalk using Docker, and setup SSL for secure connections with AWS Certificate Manager.

- **Docker**
- **AWS CLI**
- **EB CLI**
- **AWS Certificate Manager (ACM)**
- **GitHub**
- **Python (Flask)**
- **Bash**
